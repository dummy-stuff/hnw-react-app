## Getting started
* Clone the repo

    ```bash
    git clone https://github.com/student-ambassadors-futo/hnw-react-frontend.git
    ```

* Install dependencies

    ```bash
    yarn
    ```

* Run the dev server

    ```bash
    yarn dev
    ```
